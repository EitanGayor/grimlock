case class Quantile[S <: Position with ExpandablePosition, R <: Position with ExpandablePosition, W](probs: List[Double], count: Extract[S, W, Long], min: Extract[S, W, Double], max: Extract[S, W, Double], quantiser: Quantiser, name: String = "%1$f%%") extends WindowWithValue[S, R, window.Quantile.S.M] with Product with Serializable

Compute sample quantiles.

probs

List of probabilities; values must lie in (0, 1).

count

Function that extracts the count value statistics from the user provided value.

min

Function that extracts the minimum value statistics from the user provided value.

max

Function that extracts the maximum value statistics from the user provided value.

quantiser

Function that determines the quantile indices into the order statistics.

name

Pattern for the name. Use %1$s for the probability correpsponding to the quantile.
