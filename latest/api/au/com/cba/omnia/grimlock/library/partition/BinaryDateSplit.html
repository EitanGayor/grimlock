case class BinaryDateSplit[P <: Position, S](dim: Dimension, date: Date, left: S, right: S, codec: DateCodec) extends Partitioner[P, S] with Product with Serializable

Binary partition based on the date of a coordinate.

dim

The dimension to partition on.

date

The date around which to split.

left

The identifier for the left partition.

right

The identifier for the right partition.

codec

The date codec used for comparison.

Source
Partitioners.scala
Note

The position is assigned to the left partition if it is less or equal to the date value, to right otherwise.
