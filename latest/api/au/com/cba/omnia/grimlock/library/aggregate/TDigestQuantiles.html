TDigestQuantiles

Related Doc: package aggregate

case class TDigestQuantiles[P <: Position, S <: Position with ExpandablePosition, Q <: Position](probs: List[Double], compression: Double, name: FromSelectedAndOutput[S, Double, Q], filter: Boolean = true, nan: Boolean = false)(implicit ev: PosExpDep[S, Q]) extends Aggregator[P, S, Q] with PrepareDouble[P] with Product with Serializable

Compute approximate quantiles using a t-digest.

probs

The quantile probabilities to compute.

compression

The t-digest compression parameter.

name

Names each quantile output.

filter

Indicates if only numerical types should be aggregated. Is set then all categorical values are filtered prior to aggregation.

nan

Indicator if 'NaN' string should be output if the reduction failed (for example due to non-numeric data).
