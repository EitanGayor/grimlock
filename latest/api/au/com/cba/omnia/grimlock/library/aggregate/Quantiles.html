Quantiles

case class Quantiles(quantiles: Int, strict: Boolean = true, name: String = "quantile.%1$d=%2$2.3f") extends Aggregator with Prepare with PresentMultiple with StrictAggregate with Product with Serializable

Compute quantiles using nearest rank method.

quantiles

Number of quantiles to compute.

strict

Indicates if strict data handling is required. If so then any non-numeric value fails the reduction. If not then non-numeric values are silently ignored.

name

Name format for the quantiles. Usage of a %1$d in the name will be substituded with percentile index, while %2$f will be substituted by the percentage.

    Note

    An Option is used in the return type to allow aggregators to be selective in what content they apply to. For example, computing the mean is undefined for categorical variables. The aggregator now has the option to return None. This in turn permits an external API, for simple cases, where the user need not know about the types of variables of their data.
