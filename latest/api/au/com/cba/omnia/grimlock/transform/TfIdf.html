TfIdf

case class TfIdf(dim: Dimension, state: String = "idf", suffix: String = "") extends Transformer with PresentAsDoubleWithValue with Product with Serializable

Create tf-idf values.

dim

Dimension for which to create boolean term frequencies.

state

Name of the field in user supplied value which is used as the inverse document frequency.

suffix

Suffix for the new name of the coordinate at dim.

Source
Transformers.scala
Note

The returned position will have a string value at index dim. The value of the coordinate is the string representation of the original coordinate at dim with the suffix appended.

,

Tf-idf is only applied to numerical variables.
