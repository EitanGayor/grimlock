Binarise

case class Binarise(dim: Dimension, separator: String = "=") extends Transformer with PresentAndWithValue with Product with Serializable

Binarise categorical variables.

dim

Dimension for for which to create binarised variables.

separator

Sparator between position and content.

Source
Transformers.scala
Note

The returned position will have a string value at index dim. The value of the coordinate is the string representation of the original coordinate at dim with content value appended (separated by separator).

,

Binarisation is only applied to categorical variables.
