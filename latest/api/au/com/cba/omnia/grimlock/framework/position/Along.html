trait Along[P <: Position with ReduceablePosition] extends Slice[P] with Mapable[P]

Indicates that the selected coordinates are all except the one indexed by dimension. In other words, when a groupBy is performed, it is performed using a Position (type ReduceablePosition.L) consisting of all coordinates except that at index dimension.
