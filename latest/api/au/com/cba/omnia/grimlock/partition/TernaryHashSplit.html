TernaryHashSplit

case class TernaryHashSplit[S](dim: Dimension, lower: Int, upper: Int, left: S, middle: S, right: S, base: Int = 100)(implicit evidence$2: Ordering[S]) extends Partitioner with Assign with Product with Serializable

Ternary partition based on the hash code of a coordinate.

dim

The dimension to partition on.

lower

The lower ternary split ratio (relative to base).

upper

The upper ternary split ratio (relative to base).

left

The identifier for the left partition.

middle

The identifier for the middle partition.

right

The identifier for the right partition.

base

The base for the ratio.

Source
Partitioners.scala
Note

The position is assigned to the partition left if it is less or equal to lower, middle if it is less of equal to upper or else to right.

,

The hash code modulo base is used for comparison with lower/upper.
