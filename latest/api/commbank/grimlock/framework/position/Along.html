Along

Related Doc: package position

case class Along[L <: Nat, P <: Nat, D <: Nat](dimension: D)(implicit evidence$2: ToInt[D], ev1: Aux[P, L], ev2: LTEq[D, P]) extends Slice[P] with Product with Serializable

Indicates that the selected coordinates are all except the one indexed by dimension. In other words, when a groupBy is performed, it is performed using a Position[L] consisting of all coordinates except that at index dimension.

dimension

Dimension of the coordinate to exclude.
