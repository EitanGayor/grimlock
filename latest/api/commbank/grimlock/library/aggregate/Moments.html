Moments

Related Doc: package aggregate

case class Moments[P <: Nat, S <: Nat, Q <: Nat](mean: FromPosition[S, Q], sd: FromPosition[S, Q], skewness: FromPosition[S, Q], kurtosis: FromPosition[S, Q], biased: Boolean = false, excess: Boolean = false, filter: Boolean = true, strict: Boolean = true, nan: Boolean = false) extends Aggregator[P, S, Q] with MomentsPrepareReduce[P, S, Q] with Product with Serializable

Moments of a distribution.

mean

The name for the mean of the distibution

sd

The name for the standard deviation of the distibution

skewness

The name for the skewness of the distibution

kurtosis

The name for the kurtosis of the distibution

biased

Indicates if the biased estimate should be returned.

excess

Indicates if the kurtosis or excess kurtosis should be returned.

filter

Indicates if only numerical types should be aggregated. Is set then all categorical values are filtered prior to aggregation.

strict

Indicates if strict data handling is required. If so then any non-numeric value fails the reduction. If not then non-numeric values are silently ignored.

nan

Indicator if 'NaN' string should be output if the reduction failed (for example due to non-numeric data).
