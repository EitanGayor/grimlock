CountMapHistogram

Related Doc: package aggregate

case class CountMapHistogram[P <: Nat, S <: Nat, Q <: Nat](name: FromSelectedAndContent[S, Q], filter: Boolean = true)(implicit ev: GT[Q, S]) extends Aggregator[P, S, Q] with Product with Serializable

Compute histograms using a CountMap.

name

Names each histogram value output.

filter

Indicates if only categorical types should be aggregated. Is set then all numeric values are filtered prior to aggregation.

Source
Aggregators.scala
Note

Only use this if all distinct values and their counts fit in memory.
