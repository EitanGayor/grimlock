HashSplit

Related Doc: package partition

case class HashSplit[D <: Nat, P <: Nat, I](dim: D, ranges: Map[I, (Int, Int)], base: Int = 100)(implicit evidence$3: ToInt[D], ev: LTEq[D, P]) extends Partitioner[P, I] with Product with Serializable

Partition based on the hash code of a coordinate.

dim

The dimension to partition on.

ranges

A Map holding the partitions and hash code ranges (relative to base) for each partition.

base

The base for hash code.

Source
Partitioners.scala
Note

The hash code modulo base is used for comparison with the range. While a position falls in a range if it is (strictly) greater than the lower value (first value in tuple) and less or equal to the upper value (second value in tuple).
